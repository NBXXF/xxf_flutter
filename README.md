XXF框架集合
[![Pub Package](https://img.shields.io/pub/v/xxf_flutter.svg)](https://pub.dev/packages/xxf_flutter)

## Features

1. json框架
2. lifecycle框架
3. http框架
4. 路由
5. 性能框架
6. 基础组件,loading,toast
7. 桌面端[窗口控制,系统托盘,context_menu]
8. 权限
9. 日志
10. 自适应布局
11. 缓存key-value/data-base
12. 依赖注入
13. 多环境配置 flutter_dotenv
